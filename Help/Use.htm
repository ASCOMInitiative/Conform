.NET Language Drivers

The following lists how to use IConform in a VB project, if you use another language, hopefully you will be able to translate these instructions :)

  • add the IConform reference to you project, it's in the .NET tab in Visual Studio.
  • add a line to your telescope driver class file:

    Imports Conform

  • add a line after the Implements ITelescope line:

    Implements IConform

  • add the following lines to you Telescope class:

    Public ReadOnly Property ConformErrors() As ConformErrorNumbers Implements IConform.ConformErrors

    Get

    ConformErrors = New ConformErrorNumbers(&H80040400, &H80040401, &H80040402, &H80040403)

    End Get

    End Property

    Public ReadOnly Property ConformCommands() As ConformCommandStrings Implements IConform.ConformCommands

    Get

    ConformCommands = New ConformCommandStrings("GVN", "4.10", "", "", True)

    End Get

    End Property

    Public ReadOnly Property ConformCommandsRaw() As ConformCommandStrings Implements IConform.ConformCommandsRaw

    Get

    ConformCommandsRaw = New ConformCommandStrings(":GVN#", "4.10#", "", "", True)

    End Get

    End Property

  • Customise the string and integer values to your needs

Other languages through COM

In addition to the .NET assembly, an IConform type library is registered by the installer to enable exploitation of IConform by non .NET languages through Microsoft's component object model. Late binding is also supported, but, as ever with this approach, you will need to get the parameter values right based only on the documentation provided.
