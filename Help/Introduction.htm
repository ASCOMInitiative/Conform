Introduction

Conform is a program to test ASCOM driver conformance to the relevant interface standard.

Its primary users are expected to be driver authors who will be able to excercise all aspects of their driver without having to write their own test harnesses. Conform also benefits ASCOM client authors and their customers by ensuring greater homogeneity in ASCOM standard implementation and thus more consistent behaviour and hence less client application issues requiring coding to individual driver behaviour.

Version 6.0 - Key Release Changes

  • Conform tests all members of the original Platform 5 interfaces as well as the new members of the Platform 6 interfaces
  • Conform can now be configured to run as either a a 32bit or 64bit application on a 64bit OS
  • Improved testing and diagnostics in Telescope and Camera tests to assist driver developers
  • IConform support has been removed
